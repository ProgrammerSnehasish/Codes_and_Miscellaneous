5> What is overloading constructor?

Ans- More than one constructor in a class with same name as long as each has a different list of argument this concept is known as constructor overloading and it is quite similar to function overloading.

example-->

#include<iostream.h>
using namespace.std;
class construct
{
public:
float area;
construct()
{
area=a*a;
}
construct(int a,int b)
{
area=a*b;
}
void display()
{
cout<<area<<endl;
}
};
main()
{
construct o;
construct o2(10,20);
o.display();
o2.display();
}
